# 25-bit-Carry-Select-Adder
A 25 bit implementation of a Carry Select Adder. 

The adder is divided into 5 ripple carry adders of 5 bits. It provides a good speed-to-performance ratio. There are two inputs (no carry), but it can be easily modified to add a carry.  

This was coded on Xilinix vivado, but design is portable and can be implemented anywhere. 

Stimuli (3 txt files) has been provided with the correct output as well, this can be used to test the designs for the tb.
